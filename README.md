# Mergeable Chests

https://www.curseforge.com/minecraft/mc-mods/mergeable-chest

Mergeable Chests adds a new chest block that connects with matching nearby Mergeable Chests, creating one larger chest-shaped storage unit instead of a row of separate inventories.

## Features

- Merge adjacent Mergeable Chests into a wider, deeper, or taller storage block.
- Open the merged chest from any connected part and manage the combined inventory in one container.
- Uses a continuous chest model with an animated lid instead of looking like many separate chests.
- Convert between a vanilla Chest and a Mergeable Chest with shapeless crafting.

## How To Use

1. Craft a vanilla Chest into a Mergeable Chest using shapeless crafting.
2. Place Mergeable Chests next to each other with the same facing direction.
3. Right-click any connected part to open the shared inventory.
4. Craft a Mergeable Chest back into a vanilla Chest if you want to convert it back.

## Compatibility

- Requires Minecraft Forge.
- Available for Minecraft 1.20.1 and 1.19.2.
- Install on both the client and server when playing multiplayer.
